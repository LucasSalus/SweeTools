# 🌐 SweetTools - Central de Ferramentas

O **SweetTools** é uma central de automação desenvolvida para simplificar tarefas de escritório, como geração de QR Codes e conversão de dados.

## ✨ Funcionalidades
* **Gerador de QR Code**: Criação de arquivos PNG a partir de URLs com destino personalizável.
* **Conversor CSV para Word**: Transforma planilhas em documentos Word formatados em modo paisagem, ideal para grandes volumes de dados.
* **Interface Moderna**: Construída com CustomTkinter para uma experiência de usuário limpa.

## 🚀 Como usar
1. Baixe o código do repositório.
2. Instale as dependências: `pip install -r requirements.txt`.
3. Execute o arquivo principal: `python Gestor.py`.
